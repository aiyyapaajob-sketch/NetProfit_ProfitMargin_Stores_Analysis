\# Branch Net Profit \& Margin Analysis



\## Overview

This project compares three store branches using:

\- \*\*Net Profit ($)\*\* = Total Sales − Operating Cost  

\- \*\*Profit Margin (%)\*\* = (Net Profit ÷ Total Sales) × 100  



\## Data

| Branch   | Net Profit ($) | Profit Margin (%) |

|----------|----------------|-------------------|

| Branch A | 13,000         | 28.89%            |

| Branch B | 17,500         | 45.45%            |

| Branch C | 4,000          | 7.69%             |



\## Insights

\- \*\*Branch B\*\*: Strongest performer  

\- \*\*Branch A\*\*: Moderate, queues noted  

\- \*\*Branch C\*\*: High sales but weak efficiency
